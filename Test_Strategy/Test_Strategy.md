# Test Strategy

## 1. Purpose
The purpose of this document is to define the testing approach for the Kapital Bank website.

## 2. Scope
Testing includes:
- Main page
- Login page
- Cards section
- Credits section

## 3. Test Types
- Functional testing
- UI testing
- Regression testing
- Smoke testing

## 4. Test Environment
- Browser: Chrome, Firefox
- OS: Windows 10

## 5. Entry and Exit Criteria
Entry: Test environment is ready  
Exit: All critical bugs are fixed
